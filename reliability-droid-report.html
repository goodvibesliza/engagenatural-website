Executive Summary

Local build succeeds, but Netlify failed due to CI environment drift and install flags. Netlify wasn’t pinned to Node 20 and used a frozen install against cached modules. We aligned Node to 20 across toolchain files and switched to a clean pnpm install.

Root Cause Analysis

  • Primary Hypothesis: Netlify used --frozen-lockfile with no explicit Node pin, causing install failures with cached deps.
  • Contributing Factors: .nvmrc targeted Node 22 while .node-version was 20; Netlify lacked NODE_VERSION=20, leading to drift.
